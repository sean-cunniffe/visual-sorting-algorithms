Merge sort works by splitting a list into two sub-lists. This process happens with both sub-lists recursively till a sublist is only one element. The now one element sublist is merged with the other one element sublist. As merged lists are already sorted, merging the next lists is efficient.

Dividing List into sublist Merging sub-lists together Image source: https://www.interviewbit.com/tutorial/merge-sort-algorithm/

Bubble sort works by starting at the first element and comparing it to the next element. If the first element is larger, the two elements are swapped and we continue with the next element. If the second element is larger, we use that element to compare with the next element. This sorting algorithm is one of the simplest sorting algorithms.

Bubble sort example Image source: https://www.productplan.com/glossary/bubble-sort/

Selection sort works by finding the smallest element and swapping it with the first element. It then finds the next smallest number and swaps it with the second element, and so on. As the sorting algorithm goes on the unsorted list gets smaller and faster to transverse. This sorting algorithm is preferable when writing to memory is more expensive than reading.

Selection sort example Image source: https://www.hackerearth.com/practice/algorithms/sorting/selection-sort/tutorial/

Quick Sort works by selecting a pivot and moving all smaller values to the left and larger values to the right. As quick sort is recursive, we do the same thing with the values on the left side of the pivot and the values on the right side of the pivot
